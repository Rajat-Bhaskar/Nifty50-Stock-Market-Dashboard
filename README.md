# 📊 NIFTY 50 Stock Market Analytics Dashboard

A fully interactive **Power BI dashboard** built using NIFTY 50 historical stock market data. This project provides deep insights into turnover, trading volume, price trends, and top performing stocks over time.

This dashboard is designed with a **modern professional UI**, inspired by enterprise dashboards like Uber, and supports dynamic filtering and drill-down analysis.

---

# 🚀 Project Overview

This dashboard helps analyze:

• Total Turnover (₹ Crores)
• Total Volume traded
• Top Performing Stocks
• Turnover trends by Year
• Volume trends by Year
• Top 10 Stocks by Turnover
• Average, Maximum, and Total Turnover metrics

All visuals dynamically update based on selected filters.
